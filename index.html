Shift is an app to link freelancers with hiring managers in an accessible way. Eliminating the middleman, referrals are directly available on all user profiles with an average star rating and comment section. These review systems are only available after a job has been accepted; allowing only those that have interacted to review one another. This creates a greater transparency for other users when assessing a candidate or an employer.

I enjoyed using React and found its component base very comprehensible. Furthermore, my knowledge of a full stack application benefited from implementing the rating functionality using Express in the back end.

Foodlet is a food waste prevention app. It is used as a means to share food that would otherwise be disposed of. Working in a group of 3 we produced this MEAN stack app with all RESTful routes, utilising Google Maps API and AngularJS as the front-end framework.

It was our first experience with a front-end framework, what I liked most about the framework is the Two-way data binding, as it diffuses the impact after every minor data change and does way with the need for further effort with data sync in view and model.

Fruit Frenzy was my first project at General Assembly. The UI was developed by myself with inspiration taken from the game Candy Crush. Initially it was meant to be developed with the same gaming functionality as Candy Crush but as this was my first time utilising heavy JavaScript, I decided to make it a reactions game instead.
