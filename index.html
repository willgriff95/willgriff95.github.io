Fruit Frenzy
Interactive Game
Fruit Frenzy was my first project at General Assembly. The UI was developed by myself with inspiration taken from the game Candy Crush. Initially it was meant to be developed with the same gaming functionality as Candy Crush but as this was my first time utilising heavy JavaScript, I decided to make it a reactions game instead. This allowed me to manage the scalability and focus on ensuring the fundamentals were executed correctly within the time frame.
Foodlet
Web Application
Foodlet is a food waste prevention app. It is used as a means to share food that would otherwise be disposed of. Working in a group of 3 we produced this MEAN stack app with all RESTful routes, utilising Google Maps API and AngularJS as the front-end framework.
Shift
Web Application
Shift is an app to link freelancers with hiring managers in an accessible way. Eliminating the middleman, referrals are directly available on all user profiles with an average star rating and comment section. These review systems are only available after a job has been accepted; allowing only those that have interacted to review one another. This creates a greater transparency for other users when assessing a candidate or an employer.
